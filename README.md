# Awesome C# Console Toolkit

## Overview
This project is a fun and interactive C# console application. It combines a dynamic menu system with text processing features and cool ASCII art. 

## Features
* **Interactive Menu** Navigate through options easily using standard keyboard inputs.
* **Text Analysis** Count characters and modify strings on the fly.
* **ASCII Art Display** Prints fun graphics directly to the terminal using verbatim strings.
* **Color Customization** Enhances the user experience with bright and readable console colors.

## Getting Started

### Prerequisites
* .NET SDK installed on your machine
* A code editor like Visual Studio or Visual Studio Code
